# ip-2026-01

Este repositório é destinado à disciplina de Introdução à Programação (IP), do curso de Engenharia de Software da Universidade Federal de Goiás (UFG).
O conteúdo aborda fundamentos de algoritmos e sua representação, resolução de problemas por meio de soluções algorítmicas e os principais conceitos de programação, incluindo tipos de dados, variáveis, estruturas de controle (decisão e repetição) e manipulação de dados.

Este repositório reúne exercícios, listas e atividades práticas desenvolvidas ao longo da disciplina, com o objetivo de consolidar a lógica de programação e a base necessária para o desenvolvimento em engenharia de software.
